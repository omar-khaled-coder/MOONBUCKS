MOONBUCKS RETAIL ORDER MANAGEMENT SYSTEM

language: Ruby, javascript, HTML, SCSS
frame work: Rails 7.0

General Overview:

There are two actors of the system:

    Admin
    Customer

Admin can Add/Edit/Update/Edit/Delete:

    Customers
    Products
    Orders

Customer can:

    Place Order
    View Orders
    View PRoducts
